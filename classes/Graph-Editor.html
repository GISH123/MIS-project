        Class Graph-Editor

        Class defined in: js\pattern_graph.js:1

        提供一 GUI 編輯 Graph 的工具

        基本操作:

        • ctrl + left_click: 新增點
        • Hold ctrl and drag line from one vertex to another one: 修改線
        • shift + left_click: 修改點/線
        • left_click: 選取點/線
        • Select vertex/edge and DELETE: 刪除點/線
