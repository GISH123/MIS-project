### Generate document
We use [yuidoc](http://yui.github.io/yuidoc/) in this project for auto generate document.  
To generate doc, do following steps:

1. `npm install -g yuidocjs`
2. `yuidoc --theme simple js`
3. Done. the document will in `docs/`
